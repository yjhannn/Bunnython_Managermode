#Bunnython
--Manager mode
  
 1. 예약 현황
  - 며칠 간의 예약 현황을 확인할 수 있습니다.
 2. 매출 관리
  - 기업의 매출 관리를 편리하게 확인할 수 있습니다.
 3. 고객 불만사항
  - 어플로 확인하기 쉽게 고객의 소리를 담아낼 수 있습니다.
 4. 룸 서비스
  - 룸 서비스의 현황을 확인할 수 있습니다.
 5. 객실 현황
  - 현재 객실의 체크인 현황을 확인할 수 있습니다.
